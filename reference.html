Common

Visualization Options

  • sleep_time: The time, in seconds, between each step when stepping or auto solving. Defaults to 0.2.

Search

Classes with Visualizations

Searcher, AStarSearcher, SearcherMPP, DF_branch_and_bound

Included Problems

Visualization Options

  • show_edge_costs: If True, shows the cost of each edge along the line. Defaults to True.
  • show_node_heuristics: If True, show the node's heuristic value (h-value) inside the node. Defaults to False.
  • layout_method: Either "force", which is a general graph layout algorithm that works on graphs of all types, or "tree", if you know the resulting graph is always tree-shaped. Defaults to "force".

Visualization Options

  • line_width: The thickness of edges, in pixels. Defaults to 4.
