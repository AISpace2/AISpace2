For MacOS With Anaconda

With Anaconda and Script File

  1. Download Anaconda, making sure to get the Python 3+ version. The Python 2 version is not supported.
  2. You will need to install nodejs and npm if you do not have them already installed. You can find installation instructions here.
  3. Save the Command Line File to the location you want to store AISpace2.
    If you have a probelm downloading the file, right click and select "Downlad Linked File".
  4. Open the terminal then type "sh " and drag the downloaded file in to the terminal.
    The command will look like this:
    5. sh /PATH/TO/FILE/InstallAISpace2.command
  5. Execute the command.
  6. Now the Jupyter Lab will now Running.
  7. When ever you want to start the program again, Navigate to the new AISpace2 folder in your directory and run command: 
    jupyter lab

Troubleshooting

If you are using node 10 and your installation of the aispace2 labextension hangs, try to install it using node 8.

You can do this by installing nvm and running 

nvm install 8
nvm use 8
in the same folder as where you will be installing the extension.
