For MacOS With Anaconda

With Anaconda and Script File

  1. Download Anaconda, making sure to get the Python 3+ version. The Python 2 version is not supported.
  2. You will need to install Node.js if you do not have them already.
  3. Save the Install Package and the newest Release Version of AISpace2 source code to the location you want to store AISpace2.
  4. Unzip both files. Rename the source code Folder to "AISpace2". Make sure they are in the same directory.
  5. Open the terminal then type "sh " and drag the file InstallAISpace2.command in AISpace2MacInstall Directory in to the terminal.
    The command will look like this:
    6. sh /PATH/TO/FILE/InstallAISpace2.command
  6. Execute the command by pressing Enter.
  7. Now the Jupyter Lab will now Running.
  8. When ever you want to start the program again, double click the jupyterlab.command in the root directory.

Troubleshooting

If you are using node 10 and your installation of the aispace2 labextension hangs, try to install it using node 8.

You can do this by installing nvm and running 

nvm install 8
nvm use 8
in the same folder as where you will be installing the extension.
