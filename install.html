Installation

Interested in trying out AISpace2? Get started here.

Start Here!

We're happy to have you here. Follow the steps below and you'll be up-and-running in no time.

  1. Are you a UBC student or have access to a JupyterHub instance (which allows you to access Jupyter by going to a URL)? If so, you can use this extension with Jupyter online without installing anything on your computer →.
  2. Otherwise, you can install it locally if you have macOS → or Windows →. If you are running Linux →, we have a general installation procedure.
  3. Do you already have AISpace2 installed, but just want to get the latest version? This way! →

JupyterHub

Using JupyterHub means you won't have to install anything on your computer — everything takes place online.

  1. Go to ubc.syzygy.ca if you are a UBC student, and sign in with your CWL. Otherwise, go to wherever your JupyterHub instance is located and sign in.
  2. Click Start My Server when prompted.
  3. Launch a terminal instance. You can do this in the notebook dashboard by selecting New > Terminal.
  4. Now run the following commands: 
    pip install aispace2 --user
jupyter nbextension enable --py aispace2 --user

    If everything is successful, you should see the following output:

    Enabling notebook extension aispace2/extension...
        - Validating: OK

That's it! You now have Jupyter and AISpace2 successfully installed. Now download some notebooks → to learn about AI and view this extension in action.

Windows

  1. Download the latest version of Python 3. It should be version 3.3 or later.
  2. Install it, and make sure you select Add Python to Path. This is very important!
  3. Open the Command Prompt (Win + R, cmd.exe) and run: 
    pip3 install jupyter
pip3 install aispace2 --user
jupyter nbextension install --py aispace2 --user
jupyter nbextension enable --py widgetsnbextension
jupyter nbextension enable --py aispace2 --user

If everything is successful, you should see the following output:

Enabling notebook extension aispace2/extension...
- Validating: OK

That's it! You now have Jupyter and AISpace2 successfully installed. You can start Jupyter notebook by running jupyter notebook in the Command Prompt, which should launch your default web browser pointing to Jupyter notebook. Now download some notebooks → to learn about AI and view this extension in action.

macOS

If you are unfamiliar with the Python ecosystem and want the simplest installation, use Anaconda →, a Python distribution with a lot of scientific computing tools already pre-installed. Otherwise, if you prefer installing all the dependencies yourself, go here →.

With Anaconda

  1. Download Anaconda, making sure to get the Python 3+ version. The Python 2 version is not supported.
  2. Open a terminal, then execute:
    3. conda install -c conda-forge ipywidgets
pip install aispace2 --user
jupyter nbextension enable --py aispace2 --user

    You should see the following if the installation was successful:

    Enabling notebook extension aispace2/extension...
    - Validating: OK

That's it! You now have Jupyter and AISpace2 successfully installed. You can start Jupyter notebook by running jupyter notebook in a terminal, which should launch your default web browser pointing to Jupyter notebook. Now download some notebooks → to learn about AI and view this extension in action.

Without Anaconda

By default, macOS contains Python 2.7 (at least on Sierra, 10.12). You will need Python 3.3+ for it to work. Try running python3 --version in the terminal to see if you've already installed it. If it returns 3.3+, skip to the next step. If not, or it errors out, you will need to install Python 3:

  1. If you have homebrew:
    • Run brew install python3 or, if you already have Python 3 installed but need the latest version, brew upgrade python3
  2. Otherwise:
    • Download the installer from the Python website and install it

Then, confirm that pip (the Python package manager) is also installed (which it should be, as it is usually installed alongisde Python). Run pip3 --version and it should output something like:

pip 9.0.1 from /usr/local/lib/python3.6/site-packages (python 3.6)

You also want to make sure pip is updated: run pip3 install --upgrade pip. Your pip3 --version should be >9.0.0. If you encounter any problems with pip, refer to the guide here for further instructions.

All that is left to do is to install Jupyter and enable our extension:

pip3 install jupyter
pip3 install aispace2 --user
jupyter nbextension enable --py aispace2 --user
jupyter nbextension enable --py widgetsnbextension

If everything is successful, you should see the following output:

Enabling notebook extension aispace2/extension...
    - Validating: OK

That's it! You now have Jupyter and AISpace2 successfully installed. You can start Jupyter notebook by running jupyter notebook in a terminal, which should launch your default web browser pointing to Jupyter notebook. Now download some notebooks → to learn about AI and view this extension in action.

Upgrading

Already have AISpace2? No matter which operating system you use, or what method you used to install it, updating AISpace2 is as simple as running one line in the command prompt or terminal: 

pip3 install aispace2 --user --upgrade
