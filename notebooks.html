Notebooks

Code, text, and visualization-filled documents to teach AI.

Once you have Jupyter and AISpace2 installed, you can download some of the notebooks below to learn and visualize many AI concepts. These notebooks provide a easy way to explore concepts and view available functionality without having to create your own. We recommend you start with these notebooks and customize them as you see fit.

If this is your first time using Jupyter, we recommend taking a look at the following links to quickly get up to speed:
  • Notebook Basics
  • UI Overview
  • Running Code

Available Notebooks

Search
  • Simple & Advanced Search

    Includes depth-first search, A* search (with multiple path pruning), and branch & bound

Constraint Satisfaction Problems (CSPs)
  • Solving CSPs using Search

    Pose a CSP as a search problem and use the search strategies you've already learnt

  • CSPs using Arc Consistency

    Simplify constraint networks to potentially find a solution faster

  • Stochastic Local Search (SLS)

    Solve large or even infinite CSPs by moving towards solutions and incorporating randomness

Planning
  • Forward Planning

    Find a plan by searching through states and actions

  • Regression Planning

    Find a plan by searching through subgoals and actions

  • Planning as CSPs

    Represent a planning problem as a CSP by unrolling it over a certain number of steps
