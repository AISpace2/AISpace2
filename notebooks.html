Notebooks

Code, text, and visualization-filled documents to teach AI.

Once you have Jupyter and the AISpace2 extension installed, you can download some of the notebooks below to learn and visualize many AI concepts. These notebooks provide an easy way to explore concepts and learn at your own pace. We highly recommend using a modern and up-to-date browser for the best experience, such as the latest versions of Edge, Firefox, Chrome or Safari.

If this is your first time using Jupyter, we recommend taking a look at the following links to quickly get up to speed:

  • Notebook Basics
  • UI Overview
  • Running Code

Available Notebooks

Search
  • Simple & Advanced Search

    Includes depth-first search, A* search (with multiple path pruning), and branch & bound

Constraint Satisfaction Problems (CSPs)
  • Solving CSPs using Search

    Pose a CSP as a search problem and use the search strategies you’ve already learnt

  • CSPs using Arc Consistency

    Simplify constraint networks to potentially find a solution faster

  • Stochastic Local Search (SLS)

    Solve large or even infinite CSPs by moving towards solutions and incorporating randomness

Planning
  • Forward Planning

    Find a plan by searching through states and actions

  • Regression Planning

    Find a plan by searching through subgoals and actions

  • Planning as CSPs

    Represent a planning problem as a CSP by unrolling it over a certain number of steps

If the file doesn’t download automatically after clicking the link, and you are redirected to another page instead, you should download these notebooks by right-clicking the link and choosing Save As. Some web browsers have the tendency to append a .txt file extension to your notebooks when you do this — if that’s the case, you’ll have to rename the file and remove it yourself. You can do this either by using your file manager, or after uploading the notebook to Jupyter in the next step.

To use these notebooks, open Jupyter Notebook (by running jupyter notebook in a terminal if you aren’t using JupyterHub) and upload the notebook by going to the notebook dashboard, selecting Upload and choosing the notebook you just downloaded.

Then, simply open the notebook and run the cells within. Congratulations! AISpace2 is now running. Feel free to check out the reference if you need some help with all the options.
