For Windows without Anaconda

From Scratch

  • If you know that you don't have Python 3.3+ installed, go here →.
  • Otherwise, if you know you have Python 3.3+ installed already, you can confirm that it is installed correctly by opening the Command Prompt (Win + R, cmd.exe) and running 
    python -V
    It should output a line similar to this: 
    Python 3.6.3

    If so, continue with the installation here →. If it instead prints a line like this, 

    'python' is not recognized as an internal or external command, operable program or batch file.

    then Python was not added correctly to the path. You can fix this by either following the instructions here, re-running the Python installer for the exact same Python version you have (select Modify, make sure pip is selected on the next screen, tick Add Python to environment variables on the screen after that, then click Install), or installing Python again from scratch →.

Installing Python

  1. Download the latest version of Python 3. It should be version 3.3 or later.
  2. Install it, and make sure you select Add Python to Path. This is very important!

Installing AISpace2

  1. Once Python is installed, you should make sure your version of pip is up-to-date: 
    python -m pip install --upgrade pip
  2. Then install Jupyter and AISpace2: Open the Command Prompt (Win + R, cmd.exe) and run: 
    pip3 install jupyter
pip3 install jupyterlab
pip3 install aispace2 --user
  3. You will need to install nodejs and npm if you do not have them already installed. You can find installation instructions here.
  4. After that, you can install the lab extension by running the following: 
    jupyter labextension install aispace2
jupyter labextension install @jupyter-widgets/jupyterlab-manager
  5. In the folder where you would like to save the notebooks, run 
    git clone https://github.com/AISpace2/AISpace2.git
cd AISpace2
jupyter lab
  6. This will launch Jupyter Lab in your default browser. We recommend that you view the widgets using Chrome. After you run jupyter lab, the url at which the jupyter notebooks are running will be printed out in the terminal. You can copy paste this url into Chrome.

    7. Troubleshooting aispace2 labextension installation

    If you are using node 10 and your installation of the aispace2 labextension hangs, try to install it using node 8.

    You can do this by installing nvm and running 

    nvm install 8
nvm use 8
    in the same terminal as where you will be installing the extension.
