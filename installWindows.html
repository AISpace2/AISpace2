Install

Interested in trying out AISpace2? Get started here.


← Go Back

For Windows

  1. You will need to install Node.js if you do not have them already.
  2. Download AISpace2 souce code and unzip the file in the directory where you want to store it.
  3. Double click on the file AISpace2/installScripts/installScriptsWindows.bat and follow the instructions.

After the installation the JupyterLab will automatically open and you can select the notebooks (*.ipynb files) inside AISpace2/notebooks/ to run. In the future if you want to run AISpace2, simply run the following in the Command Prompt: 

jupyter lab

Troubleshooting

If you are using node 10 and your installation of the AISpace2 labextension hangs, try to install it using node 8.

You can do this by installing nvm and running 

nvm install 8
nvm use 8
in the same folder as where you will be installing the extension.
